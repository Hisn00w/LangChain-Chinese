## 📘 关于本项目

这是 LangChain 官方文档的中文翻译项目，旨在帮助中文用户更好地学习和使用 LangChain。

### ✨ 项目特点

- 持续同步官方文档结构，降低学习门槛
- 保留原文的技术术语与代码示例
- 提供清晰的文档导航与章节划分

## 🚀 快速开始

请从左侧目录开始阅读，推荐从以下章节入门：

- [概览](overview.md) - 了解 LangChain 是什么
