# Installation

> 官方原文：https://docs.langchain.com/docs/installation

## 环境要求

- Python 3.8+
- pip 或 conda

## 安装方式

### pip

```bash
pip install langchain
```

### conda

```bash
conda install langchain -c conda-forge
```

## 可选的集成包

```bash
# 安装所有集成
pip install langchain[all]

# 仅安装 OpenAI
pip install langchain-openai

# 仅安装 Anthropic
pip install langchain-anthropic
```
